# Changelog

## v0.1.0
- Initial release of the PRTG domain expiration sensor
- Added RDAP-first lookup and WHOIS fallback
- Added XML output compatible with PRTG
- Added basic installation, usage, and example documentation

## v0.2.0
- Added support for `.ua`, `.com.ua`, and `.dp.ua`
- Improved domain parsing and zone handling

## v0.3.0
- Added support for `.pro`, `.wine`, `.cy`, `.bg`, and `.ae`
- Improved error handling and fallback logic

## v1.0.0
- Stable release for open-source use
- Finalized structure for PRTG deployment and documentation
